# CTF Challenge Level 1

## Challenge Title: Basic Web App

### Description:
This challenge involves finding hidden elements in a basic web application. Participants need to identify a flag hidden within the HTML source code.

### Practical Steps:
1. Visit the web page.
2. Right-click and select 'View Page Source'.
3. Search for keywords like 'flag' or 'CTF'.

### Flag Format:
ctf{example_flag_1}
